TradingCardDashboard Mobile v3.0

主な変更
- 大容量JSONの保存先をlocalStorageからIndexedDBへ変更
- iPhoneの「The quota has been exceeded」エラーを解消
- ログイン済み端末では起動時にSupabaseから最新データを自動取得
- 手動アップロード／ダウンロードも継続利用可能
- 古いキャッシュを自動更新するService Worker v3

GitHub更新方法
このフォルダ内のファイルをTradingCard-Trackerリポジトリの同名ファイルへ上書きしてください。
GitHub Pagesの設定変更は不要です。
