TradingCard Dashboard Mobile

使い方
1. index.html をPCブラウザで開く
2. 「JSONを読み込む」を押す
3. PC版トレカ資産トラッカーから書き出したJSONを選ぶ

iPhoneで使う場合
- このフォルダをWeb公開した後、Safariで開く
- 共有 → ホーム画面に追加

注意
- PC版のファイルには一切触れません
- 読み込んだJSONは、このスマホ版ブラウザ内に保存されます
- データ更新時は新しいJSONを再度読み込んでください
